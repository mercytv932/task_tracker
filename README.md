This is an interactive web application built with HTML, CSS, and JavaScript, designed to help users efficiently organize and manage their academic tasks. The application features a dynamic task system that allows users to add subjects, assignments, and due dates, track completion status, and persist data using localStorage for seamless use across sessions. A responsive navigation bar, animated gradients, floating UI effects, and wave-based background animations enhance the visual experience, while real-time filtering highlights upcoming and overdue tasks for better prioritization. Additional functionality such as task completion sound feedback, task statistics tracking, and responsive layout adjustments demonstrates a strong focus on usability, state management, and modern front-end design principles.
